# Svocal

Svocal combines the power of Svelte stores with localstorage.  
This packageg is currently very early, while I use it myself I wouldn't really recomend to use it!

## Installation

```bash
npm i svocal
```

```bash
yarn add svocal
```

```bash
pnpm add svocal
```

```bash
bun add svocal
```
